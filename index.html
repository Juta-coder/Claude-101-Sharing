18:00 · Anti-Hallucination

กฎทอง — ป้องกัน Hallucination

⏱ 7 นาที
❌ Prompt อันตราย
"วิเคราะห์ผลประกอบการ Q3 ของบริษัทเรา"
Claude ไม่มีข้อมูล → สร้างตัวเลขขึ้นมาเอง = Hallucination
✅ Prompt ถูกต้อง
"นี่คือข้อมูล Q3: [ตัวเลข] — วิเคราะห์ให้หน่อย"
Claude ทำงานกับข้อมูลจริง ✓
✅ พิมพ์แบบนี้
นี่คือข้อมูล P&L ของเรา: Revenue Q1=100M, Q2=115M, Q3=108M COGS Q1=60M, Q2=68M, Q3=67M วิเคราะห์ trend และสิ่งที่น่ากังวล แสดงขั้นตอนการคำนวณด้วย
✋ ลองทำ: พิมพ์ prompt ผิด → ดู hallucination → พิมพ์ prompt ถูก → เปรียบกัน
📁
25:00 · Co-work / Project

Co-work — Claude ที่จำ context คุณได้

⏱ 12 นาที
💡 ตั้งค่าครั้งเดียว — ใช้ได้ตลอด
Claude จำ: ภาษา · หน่วยเงิน · สไตล์การตอบ · files ที่ upload ไว้
1
claude.ai → คลิก "Projects" → สร้าง Project ชื่อ "CFO Analysis"
2
เพิ่ม Project Instructions (copy จากด้านล่าง)
3
Upload ไฟล์ Excel หรือ PDF เข้า Project
4
ลองถามโดยไม่ต้องบอก context ซ้ำ — Claude จะจำเอง
⚙️ Project Instructions — copy ไปวาง
คุณคือ Senior Financial Analyst ของบริษัท [ชื่อบริษัท] - ตอบเป็นภาษาไทยเสมอ · ใช้หน่วยล้านบาท · สรุปไม่เกิน 5 bullet - แสดงขั้นตอนการคำนวณทุกครั้ง - ถ้าไม่มีข้อมูลในเอกสาร ให้บอกว่า "ไม่มีข้อมูลนี้" - อย่า assume ตัวเลขที่ไม่มีในเอกสาร
✋ ลองทำ: สร้าง Project + ตั้ง Instructions + ถามโดยไม่บอก context
📊
37:00 · Practical

ขอ Claude เขียน Excel Formula

⏱ 5 นาที
💡 ไม่ต้องเป็น programmer
แค่บอก Claude ว่าต้องการ formula อะไร → Claude เขียนให้ + อธิบาย → copy ไป paste ใน Excel ได้เลย
📝 พิมพ์ตาม
ช่วยเขียน Excel formula สำหรับ: - คำนวณ Gross Margin % โดย Revenue อยู่ใน cell B2, COGS อยู่ใน C2 - รวม error handling กรณีตัวหารเป็น 0 - อธิบายว่า formula ทำงานอย่างไร
✋ ลองทำ: ขอ formula → copy → paste ใน Excel → ทดสอบก่อนใช้จริง

42:00 · Skill

Skill — สูตรสำเร็จสำหรับงานที่ทำซ้ำ

⏱ 8 นาที
❌ ไม่มี Skill
พิมพ์ instructions ทุกครั้ง · แต่ละคนพิมพ์ต่างกัน · output ไม่สม่ำเสมอ
✅ มี Skill
เปิด Skill → ใส่ข้อมูล → output รูปแบบเดิมทุกครั้ง · ทั้งทีมใช้ได้
⚡ ตัวอย่าง Skill — Monthly Variance Report
คุณคือ Financial Analyst งาน: วิเคราะห์ Monthly Variance Report — Output: 1. Revenue Variance — จริง vs แผน + เหตุผล 2. COGS Variance — จริง vs แผน + เหตุผล 3. OpEx Variance — จริง vs แผน + เหตุผล 4. Net Profit + 3 ข้อที่ต้องติดตาม ตอบภาษาไทย หน่วยล้านบาท
✋ ลองทำ: สร้าง Skill แรกของคุณ
